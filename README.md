**View your wallets and balances on block.io**

The repository contains a program that allows you to view your balances and 
wallets at https://block.io/ through the API.

**Installation**

To install, you must download the repository and have an environment,
for example like this: IntelliJ IDEA.

**Quick Start**
You must be registered with block.io and have wallets. 
Go to the downloaded project via IntelliJ IDEA and replace the API key
with yours.
You can find the API documentation at https://block.io/docs/basic
Just run the project. 
When finished, you should see in the root folder, two files 
walletsWithBalance.csv and allWallets.csv.
